## Installation (Easy)

## Installation (Manual)
1. Add this Repository to HACS:
   - HACS > 3 dots > "Add custom repository"
   - URL: `Bitte-ein-Git/ha_kodi_helpers`
   - Type: Integration

2. Select "🍿• Kodi-Helpers".
